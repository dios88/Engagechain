    03 — PROTOCOL

    How It Works

    01
    ✍

    Human Input

    The user submits an opinion, proposal, or dispute. The GenLayer Python contract records it with a unique ID and marks it as pending.

    02
    ◉

    AI Evaluation

    The Intelligent Contract calls the LLM via gl.exec_prompt(). The AI analyzes the text and generates a structured JSON verdict.

    03
    ⬡

    Optimistic Democracy

    GenLayer validators run the same prompt locally and compare results using the Equivalence Principle. Semantically equivalent outputs are accepted.

    04
    ⛓

    On-Chain Record

    The consensus verdict is stored in the contract. Anyone can verify the opinion, AI response, and final status without centralized trust.
